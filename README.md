## 🎯 Tecnologias utilizadas

- Next.js
- TypeScript
- Styled Components 

## 🚀 Funcionalidades

- Maximizar tela
- Minimizar tela
- Fechar tela
- Minimizar menu
  
## ⚙️ Como executar

Após clonar o repositório, acesse a pasta do projeto e execute os comandos abaixo:

```sh
npm install
npm run dev
```

Acesse http://localhost:3000 para visualizar a aplicação.
